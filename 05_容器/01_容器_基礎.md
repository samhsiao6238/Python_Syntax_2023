# 容器

_`容器` 類型的認識與使用在 Python 學習中是一個很重要的環節，所以也稍微從嚴謹的角度說明了一下，看看即可。_

<br>

## 簡單說明

1. 依據官方文檔的描述，`容器` 是用來 `儲存和組織其他物件的一種物件`。
2. 所以要能稱之為 `容器`，最基本必須滿足這個要件。

<br>

## 容器的特點

1. 容器內可以沒有物件或包含一個以上的物件。
2. 容器支援成員偵測操作，即可以檢查一個物件是否包含在容器中。
3. 容器可以迭代，即可以遍歷容器中的所有元素。
4. 基本資料類型中如整數、浮點數、布林值不是容器。
5. 其他常見的容器類型包括字串、列表、元組、集合、和字典等。

<br>

## Container 容器的嚴謹定義

_可跳過_

1. Python 透過 `抽象基底類別`（Abstract Base Class, ABC）來定義 Container。
2. 在 collections.abc 模組中，有一個名為 `Container` 的 `抽象基底類別` ，這個類別為容器定義了一個抽象方法：`__contains__()` 。
3. 基於以上兩點可知，任何繼承了 `Container ABC` 並實作了 `__contains__()` 方法的類別都可以被視為一個容器。
4. `__contains__()` 方法是用來檢查容器中是否包含某個元素，它接受以 `要檢查的元素` 作為參數，並傳回一個布林值來表示該元素是否在容器中。
5. 一個容器生成的過程如下 `MyContainer` 的範例，先繼承了 `Container` 類，再實作 `__contains__()` 方法，所以它可以被視為一個容器，至此闡述了如何透過父類與實作方法來識別容器。

    ```python
    # 載入要繼承的類
    from collections.abc import Container

    # 繼承 Container
    class MyContainer(Container):
        # 再實作這個方法
        def __contains__(self, value):
            pass
   ```

## 容器的種類

### 1. 列表 List

- 類別名稱：`list`
- 列表是一個有序的集合，可以隨時添加和刪除其中的元素。

```python
lst = [1, 2, 3, 4]
```

<br>

### 2. 元組 Tuple

- 類別名稱：`tuple`
- 元組和列表類似，但元組的成員是不可修改的。

```python
t = (1, 2, 3, 4)
```

<br>

### 3. 字典 Dictionary

- 類別名稱：`dict`
- 字典是`無序的鍵值對集合` （key-value），`鍵 key` 必須是唯一的。

```python
d = {"key": "value"}
```

<br>

### 4. 集合 Set

- 類別名稱：`set`
- 集合是一個 `無序` 且 `不重複` 的元素集。

```python
s = {1, 2, 3}
```

<br>

### 5. 字串 String

- 類別名稱：`str`
- 字串是用來表示文本的數據類型。

```python
str_val = "Hello, World!"
```

<br>

### 6. 陣列 Array

- 類別名稱：`array.array`
- Python 的內置模組，需導入 `import` 方可使用，導入後可用來創建數值陣列，而陣列的所有元素都須是同一類型。
- 建立第一個參數是數據類型，其中 `i` 代表指數、`f` 代表浮點數、`u` 代表 Unicode 字符，第二個參數要傳入可迭代物件，列表、數組、字串皆可。

```python
import array
arr_1 = array.array('i', (1, 2, 3))
arr_2 = array.array('f', [1, 2, 3])
arr_3 = array.array('u', 'abc')
print(arr_1)
print(arr_2)
print(arr_3)
```
_OUTPUT_
```bash
array('i', [1, 2, 3])
array('f', [1.0, 2.0, 3.0])
array('u', 'abc')
```

<br>

### 7. 雙端隊列 Deque

- 類別名稱：`collections.deque`
- 雙端隊列是 collections 模組中的一個子類，是一種雙端隊列，允許從其頭部和尾部高效地添加和刪除元素，與列表相比，deque 提供了更快速的應用程序。

```python
from collections import deque
dq = deque([1, 2, 3, 4])
# 插入在右邊
dq.append(5)
print(dq)   # deque([1, 2, 3, 4, 5])
# 插入在左邊
dq.appendleft(0)
print(dq)   # deque([0, 1, 2, 3, 4, 5])
# 取出最上面
dq.pop()
print(dq)   # deque([0, 1, 2, 3, 4])
# 取出最左邊
dq.popleft()
print(dq)   # deque([1, 2, 3, 4])
# 拓展加入一個列表
dq.extend([5, 6, 7])
print(dq)   # deque([1, 2, 3, 4, 5, 6, 7])
# 拓展加入在左邊
dq.extendleft([-1, -2, -3]) # 特別注意是一個一個取出並插入在最左邊
print(dq)   # deque([-3, -2, -1, 1, 2, 3, 4, 5, 6, 7])
# 正值：將右邊2個轉到左邊
dq.rotate(2)
print(dq)   # deque([6, 7, -3, -2, -1, 1, 2, 3, 4, 5])
# 負值：將左邊3個轉到右邊
dq.rotate(-3)
print(dq)   # deque([-2, -1, 1, 2, 3, 4, 5, 6, 7, -3])
```

### 8. 堆 Heap

- 類別名稱：不是特定的類別，而是使用 `list` 配合 `heapq` 模組來實現。
- `堆` 是一種特殊的樹形數據結構，它的每個父節點都小於或等於其子節點，而 heapq 模組提供了一種實現堆數據結構的方式。
- 下方腳本展示如何使用 heapq 模組來操作 `堆數據結構`，包括向堆添加元素、取得堆頂元素、以及將一個普通的 list 轉換成為有效的堆結構。

```python
import heapq

# 初始化一個空堆
heap = []

# 使用 heappush 將元素加入堆
heapq.heappush(heap, 4)
heapq.heappush(heap, 2)
heapq.heappush(heap, 8)
heapq.heappush(heap, 5)

# 印出堆的內容
print(heap)  # 輸出：[2, 4, 8, 5]

# 使用 heappop 取得堆頂元素（最小的元素）並移除
print(heapq.heappop(heap))  # 輸出：2
print(heap)  # 輸出：[4, 5, 8]

# 使用 heapify 轉換 list 成為有效的堆結構
nums = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]
heapq.heapify(nums)
print(nums)  # 輸出：[1, 1, 2, 3, 3, 9, 4, 6, 5, 5, 5]
```

### 9. 計數器 Counter

- 類別名稱：`collections.Counter`
- 用於計數可哈希對象的特殊字典，是 collections 模塊中的子類。

```python
from collections import Counter

c = Counter(["蘋果", "香蕉", "蘋果", "橘子", "香蕉", "蘋果"])
print(c)  # 輸出：Counter({'蘋果': 3, '香蕉': 2, '橘子': 1})
```

<br>

### 10. 有序字典 OrderedDict

- 類別名稱：`collections.OrderedDict`
- 保持元素插入順序的字典，是 collections 模塊的子類。

```python
from collections import OrderedDict

od = OrderedDict()
od["a"] = 1
od["b"] = 2
od["c"] = 3
od["g"] = 5
od["f"] = 4
print(od)  # 輸出：OrderedDict({'a': 1, 'b': 2, 'c': 3, 'g': 5, 'f': 4})
```

<br>

### 11. 命名元組 NamedTuple

- 類別名稱：`namedtuple` 。
- 根據給定的名稱和欄位返回值是一個新的 `類型`，並依據欄位定義屬性。
- namedtuple 可以創建帶有名稱的 `元組子類` ，是 collections 模塊中的子類。

```python
from collections import namedtuple

Person = namedtuple('Person', ['name', 'age', 'gender'])
p = Person(name="John", age=30, gender="Male")
print(p.name)  # 輸出：John
print(Person.__bases__)
```

_OUTPUT_

```bash
John
(<class 'tuple'>,)
```

<br>

### 12. 默認字典 DefaultDict

- 類別名稱：`collections.defaultdict`
- collections 模塊中的 defaultdict 類型是一種當所訪問的鍵不存在時，會自動創建該鍵並將其值設定為默認值的字典。

```python
from collections import defaultdict

dd = defaultdict(int)
dd["a"] = 1
dd["b"] = 2
print(dd)
print(dd["c"])  # 輸出：0，因為默認值是0
print(dd)
```

_OUTPUT_

```bash
defaultdict(<class 'int'>, {'a': 1, 'b': 2})
0
defaultdict(<class 'int'>, {'a': 1, 'b': 2, 'c': 0})
```

<br>

### 13. 範圍 range

- 類別名稱：`range` 。
- 是一個不可變的序列型別，主要在迴圈中使用。
- 可利用來生成一系列的數字。

```python
for i in range(5):
    print(i)
```
_OUTPUT_
```bash
0
1
2
3
4
```

<br>

### 14. 凍結集合 frozenset

- 類別名稱：`frozenset`
- 不可變的集合。
- 它和 `set` 功能相似，但一旦被創建後就不能修改。

```python
fs = frozenset([1, 2, 3, 4])
print(fs)
print(type(fs))
```
_OUTPUT_
```bash
frozenset({1, 2, 3, 4})
<class 'frozenset'>
```

<br>

### 15. 字節 bytes

- `bytes` 是用來表示字節序列的。

```python
b = bytes("hello", encoding="utf-8")
print(b)
```
_OUTPUT_
```bash
b'hello'
```

<br>

### 16. 字節陣列 bytearray

- `bytearray` 是一種可變的字節序列。

```python
ba = bytearray("hello", encoding="utf-8")
ba[0] = 72  # 更改第一個字節
```

<br>

### 17. 記憶體視圖 memoryview

- `memoryview` 是一個內置的二進制序列型別，它可以用來表示其他物件的內存視圖。

```python
mv = memoryview(b"hello")
```

<br>

### 18. 工具集 itertools

- 類別名稱：不是特定的類別，而是一個模組。
- Python 的標準庫中的 `itertools` 模組提供了許多實用的工具，如 `chain`、`cycle` 和 `groupby` 等。

```python
from itertools import chain

result = chain([1, 2, 3], [4, 5, 6])
for item in result:
    print(item)
```
