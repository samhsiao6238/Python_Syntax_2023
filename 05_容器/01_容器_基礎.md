## 1. 列表 List

- 類別名稱：`list`
- 列表是一個有序的集合，可以隨時添加和刪除其中的元素。

```python
lst = [1, 2, 3, 4]
```

<br>

## 2. 元組 Tuple

- 類別名稱：`tuple`
- 元組和列表類似，但元組的元素不能修改。
```python
t = (1, 2, 3, 4)
```

<br>

## 3. 字典 Dictionary

- 類別名稱：`dict`
- 字典是無序的鍵值對集合，鍵必須是唯一的。

```python
d = {"key": "value"}
```
<br>

## 4. 集合 Set

- 類別名稱：`set`
- 集合是一個無序且不重複的元素集。

```python
s = {1, 2, 3}
```

<br>

## 5. 字串 String

- 類別名稱：str
- 字串是用來表示文本的數據類型。

```python
str_val = "Hello, World!"
```

<br>

## 6. 陣列 Array

- 類別名稱：array.array
- Python 的內置模組，需導入 `import` 方可使用，導入後可用來創建密集的數值陣列，而陣列與列表類似，但所有元素都須是同一類型。

```python
import array
arr = array.array('i', [1, 2, 3])
```

<br>

## 7. 雙端隊列 Deque

- 類別名稱：collections.deque
- 雙端隊列是 collections 模組中的一個子類，是一種雙端隊列，支持在兩端快速地添加和刪除元素。

```python
from collections import deque
dq = deque([1, 2, 3])
```

<br>

## 8. 堆 Heap

- 類別名稱：不是特定的類別，而是使用 `list` 配合 `heapq` 模組來實現。
- `堆` 是一種特殊的樹形數據結構，它的每個父節點都小於或等於其子節點，而 heapq 模組提供了一種實現堆數據結構的方式。
- 下方腳本展示如何使用 heapq 模組來操作 `堆數據結構`，包括向堆添加元素、取得堆頂元素、以及將一個普通的 list 轉換成為有效的堆結構。

```python
import heapq

# 初始化一個空堆
heap = []

# 使用 heappush 將元素加入堆
heapq.heappush(heap, 4)
heapq.heappush(heap, 2)
heapq.heappush(heap, 8)
heapq.heappush(heap, 5)

# 印出堆的內容
print(heap)  # 輸出：[2, 4, 8, 5]

# 使用 heappop 取得堆頂元素（最小的元素）並移除
print(heapq.heappop(heap))  # 輸出：2
print(heap)  # 輸出：[4, 5, 8]

# 使用 heapify 轉換 list 成為有效的堆結構
nums = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]
heapq.heapify(nums)
print(nums)  # 輸出：[1, 1, 2, 3, 3, 9, 4, 6, 5, 5, 5]
```

## 9. 計數器 Counter

- 類別名稱：collections.Counter
- 用於計數可哈希對象的特殊字典，是 collections 模塊中的子類。

```python
from collections import Counter

c = Counter(["蘋果", "香蕉", "蘋果", "橘子", "香蕉", "蘋果"])
print(c)  # 輸出：Counter({'蘋果': 3, '香蕉': 2, '橘子': 1})
```


<br>

## 10. 有序字典 OrderedDict

- 類別名稱：collections.OrderedDict
- 保持元素插入順序的字典，是 collections 模塊的子類。

```python
from collections import OrderedDict

od = OrderedDict()
od["a"] = 1
od["b"] = 2
od["c"] = 3
od["g"] = 5
od["f"] = 4
print(od)  # 輸出：OrderedDict({'a': 1, 'b': 2, 'c': 3, 'g': 5, 'f': 4})
```

<br>

## 11. 命名元組 NamedTuple

- 類別名稱：`namedtuple` 。
- 根據給定的名稱和欄位返回值是一個新的`類型`，並依據欄位定義屬性。
- namedtuple 可以創建帶有名稱的 `元組子類` ，是 collections 模塊中的子類。 

```python
from collections import namedtuple

Person = namedtuple('Person', ['name', 'age', 'gender'])
p = Person(name="John", age=30, gender="Male")
print(p.name)  # 輸出：John
print(Person.__bases__)
```
_OUTPUT_
```bash
John
(<class 'tuple'>,)
```

<br>

## 12. 默認字典 DefaultDict

- 類別名稱：collections.defaultdict
- collections 模塊中的 defaultdict 類型是一種當所訪問的鍵不存在時，會自動創建該鍵並將其值設定為默認值的字典。

```python
from collections import defaultdict

dd = defaultdict(int)
dd["a"] = 1
dd["b"] = 2
print(dd)
print(dd["c"])  # 輸出：0，因為默認值是0
print(dd)
```
_OUTPUT_
```bash
defaultdict(<class 'int'>, {'a': 1, 'b': 2})
0
defaultdict(<class 'int'>, {'a': 1, 'b': 2, 'c': 0})
```

<br>

## 13. 範圍 `range`

- 類別名稱：`range` 。
- 是一個不可變的序列型別，主要在迴圈中使用。
- 可利用來生成一系列的數字。

```python
for i in range(5):
    print(i)
```

<br>

## 14. 凍結集合 `frozenset`

- 類別名稱：frozenset
- 不可變的集合。
- 它和 `set` 功能相似，但一旦被創建後就不能修改。

```python
fs = frozenset([1, 2, 3, 4])
```

<br>

b = bytes("hello", encoding="utf-8")

- `bytearray` 是一種可變的字節序列。

## 17. 記憶體視圖 `memoryview`

- `memoryview` 是一個內置的二進制序列型別，它可以用來表示其他物件的內存視圖。

```python
mv = memoryview(b"hello")
```

<br>

## 18. 工具集`itertools`

- 類別名稱：不是特定的類別，而是一個模組。
- Python 的標準庫中的 `itertools` 模組提供了許多實用的工具，如 `chain`、`cycle` 和 `groupby` 等。

```python
from itertools import chain

result = chain([1, 2, 3], [4, 5, 6])
for item in result:
    print(item)
```
